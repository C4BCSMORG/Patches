DE
Die Softphone-Clients bekommen vom Server keinen STUN-Server zur Verfügung gestellt.

EN
The softphone clients are not provided with a STUN server by the server.
</description>

DE
Automatische Patch Installation:
1. Beenden Sie den XPhone Connect Server Dienst und zugehörige Dienstprogramme
2. Starten Sie die Batch Datei "Install_Patch.Bat"
3. Abschließend starten Sie den XPhone Connect Server Dienst

Manuelle Patch Installation:
1. Beenden Sie den XPhone Connect Server Dienst und zugehörige Dienstprogramme
2. Kopieren Sie den Inhalt aus dem "bin" Verzeichnis des Patches in das XPhone Server Hauptverzeichnis und ersetzen Sie die bestehenden Dateien
3. Abschließend starten Sie den XPhone Connect Server Dienst


EN
Automatic patch installation:
1. Stop the XPhone Connect Server service and associated utilities
2. Start the batch file "Install_Patch.Bat"
3. Finally, start the XPhone Connect Server service

Manual patch instlalation:
1. Stop the XPhone Connect Server service and associated utilities
2. Copy the content of the "bin" directory of the patch into the XPhone Server root directory and replace the existing files
3. Finally, start the XPhone Connect Server service
